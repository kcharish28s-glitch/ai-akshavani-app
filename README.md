# 📱 AI Akshavani — Telugu News App

> Way2News style Telugu news app with AI-powered content management

- Telugu category tabs
- Breaking news ticker
- Admin panel — manual news content management
- User & Admin login roles

## 🛠️ Tech Stack
- HTML + CSS + JavaScript
- VS Code + Claude Code
- localStorage (no backend required)
- Mobile-first responsive design
